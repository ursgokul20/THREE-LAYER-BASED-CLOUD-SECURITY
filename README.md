# ☁️ Three Layer Based Cloud Security

This repository documents the concept of **Three Layer Based Cloud Security**, a defense-in-depth approach to securing cloud environments.

---

## 🔐 1. Infrastructure Security Layer
Focus: Protecting physical servers, virtualization, and network components.

**Controls:**
- Firewalls, IDS/IPS
- Secure hypervisors
- Network segmentation
- DDoS protection

---

## 🛡️ 2. Platform Security Layer
Focus: Securing cloud services such as compute, storage, and databases.

**Controls:**
- Identity & Access Management (IAM)
- Encryption (data at rest & in transit)
- Secure APIs
- Patch management

---

## 📲 3. Application Security Layer
Focus: Safeguarding end-user applications hosted in the cloud.

**Controls:**
- Secure coding practices
- Web Application Firewall (WAF)
- Vulnerability scanning
- Continuous monitoring & logging

---

## 📌 Key Points
- **Defense in Depth**: Each layer adds protection, reducing risk of single point failure.  
- **Shared Responsibility Model**: Cloud provider secures infrastructure, customer secures apps & data.  
- **Compliance**: Align with ISO 27001, NIST SP 800-30, GDPR, etc.
